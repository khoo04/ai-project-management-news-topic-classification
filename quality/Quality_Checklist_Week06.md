# Quality Definition (Week 06)
- **Accuracy target:** ≥ 85% 
- **Performance expectations:** 
  - *Achieves* ≥ 90% model accuracy  
  - *Inference* speed ≤ 200ms per prediction
- **User interface/usability expectations:** Clean Layout, Easy Navigation and consistent styling
- **Data quality requirements:** Cleaned dataset, no duplicates, minimal missing values, correct labelling
